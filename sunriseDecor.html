Sunrise Decor

Furniture eCommerce Site

Project Overview

In this project, I crafted a website using HTML and CSS, focusing on the display of furniture items such as chairs, couches, and tables. The design incorporates a navigation system with tabs for each furniture category, ensuring a user-friendly shopping experience. Using CSS grid layout, the website achieves an organized presentation, allowing users to easily explore the products this store offers.

I utilized Flexbox in the presentation of furniture items, with each card showcasing product details. The Flexbox cards provide a responsive and dynamic layout, adapting to different screen sizes. This responsive design is further enhanced through media queries, ensuring that the eCommerce website seamlessly adjusts to fit any device, whether it's a desktop, tablet, or smartphone. The combination of CSS Grid and Flexbox not only enhances the visual appeal of the website but also contributes to a user-centric design, creating an immersive and enjoyable browsing experience.
